# 接口列表和接口详情优化计划

## 目标

1. **中间区域（接口列表）**：
   - 删除重复的WebSocket标签（蓝色字体那个）
   - 当前接口类型标签的两个类型添加颜色区分和图标区分

2. **右侧区域（接口详情）**：
   - 将接口类型样式改为居左对齐

## 修改内容

### 1. 修改接口列表组件 (InterfaceListView.vue)

**修改位置**：第53-60行，接口标签区域

**修改内容**：
- 删除重复的WebSocket标签：第54-56行的v-if="intf.is_websocket"的WebSocket标签
- 为当前接口类型标签添加颜色区分：
  - 当`is_websocket`为true时，使用type="primary"，图标<i class="el-icon-link"></i>
  - 当`is_websocket`为false时，使用type="success"，图标<i class="el-icon-document"></i>

**修改后效果**：
- 接口列表中只显示一个当前接口类型标签
- 不同类型的接口标签有明显的颜色区分和图标区分
- 界面更简洁，信息更清晰

### 2. 修改接口详情组件 (InterfaceDetailView.vue)

**修改位置**：第84-94行，接口类型开关区域

**修改内容**：
- 将text-align从center改为left
- 调整样式，使其左对齐显示
- 保持label和开关的间距不变

**修改后效果**：
- 接口类型开关居左对齐，更符合用户阅读习惯
- 界面布局更合理
