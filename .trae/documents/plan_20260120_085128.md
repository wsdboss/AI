# 修复allMockEnabled重复定义警告

## 1. 错误原因
- **错误信息**：`[Vue warn]: The computed property "allMockEnabled" is already defined in data.`
- **根本原因**：在 `InterfaceListView.vue` 文件中，`allMockEnabled` 属性同时在 `data()` 和 `computed` 中定义
- **错误位置**：
  - `data()` 中：第106行 `allMockEnabled: false`
  - `computed` 中：第143行 `allMockEnabled: { get() { ... }, set(value) { ... } }`

## 2. 修复方案
删除 `data()` 中的 `allMockEnabled` 属性定义，只保留 `computed` 中的计算属性定义：

```javascript
// 修复前 - data() 中定义
data() {
  return {
    // ...其他属性
    allMockEnabled: false, // 这里是重复定义
    // ...其他属性
  }
}

// 修复后 - 删除了 data() 中的重复定义
data() {
  return {
    // ...其他属性
    // 已删除 allMockEnabled 属性
    // ...其他属性
  }
}
```

## 3. 修复效果
- ✅ 消除 Vue 警告
- ✅ 保留 `computed` 中的计算属性功能
- ✅ 不影响组件的正常功能

## 4. 技术要点
- Vue 不允许同一个属性名同时出现在 `data` 和 `computed` 中
- `computed` 中的 `allMockEnabled` 是一个带有 getter 和 setter 的计算属性，用于控制所有接口的 Mock 服务状态
- 删除 `data()` 中的重复定义后，Vue 将正确使用 `computed` 中的计算属性

## 5. 文件修改列表
- `frontend/views/InterfaceListView.vue` - 删除 data() 中的 allMockEnabled 属性定义
